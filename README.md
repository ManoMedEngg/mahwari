## About the project

**mahwari** is a privacy-first, aesthetic monthly cycle tracker designed to empower you with knowledge about your body. The application has the necessary features that help you track your cycle and health habits. The app shows:

- 🗓️ **Possible next period date**
- 🔄 **Current cycle day & Phase (Follicular, Luteal, etc.)**
- 👶 **Chance of getting pregnant (Fertility Window)**
- 🥚 **Number of days before ovulation**
- 📊 **History of previous cycles** & average cycle/period length
- 💧 **Daily Water Intake** tracking
- 🧘 **Exercise Guide** tailored for menstrual health

It also tracks the current phase of the cycle and displays information about this phase and the symptoms that a woman may feel during this specific phase.

All data is stored **locally on your device**. No cloud, no tracking, complete privacy.

## 🛠️ How to build

You can run this project locally with a few simple commands.

### Prerequisites
- Node.js (v18+)
- npm

### Installation
```bash
# Clone the repository
git clone https://github.com/ManoMedEngg/mahwari.git

# Install dependencies
npm install

# Run development server
npm run dev
```

## 📄 Algorithms and calculations

The app uses standard medical averages for predictions:
- **Follicular Phase**: Days 6-13
- **Ovulation**: 14 days before the next expected period.
- **Fertility**: High during Ovulation +/- 2 days.
